# MITI
전국 각지에 있는 농구 경기장을 확인 할 수 있고 경기를 생성해 모집할 수 있습니다.<br/>
건강한 생활 체육인이 될 수 있는 전용 앱<br/>
가까운 동네 사람들과 함께 농구 경기를 진행해보세요!<br/>
